# Action-box

## Tag Name

`v-action-box`

## Usage

```html
<v-action-box v-model="value" :maskCloseable="false">
  <!-- slot 内容区 -->
</v-action-box>
```

## Options

属性名   |    类型    |    默认值    |   说明
----    | ----      | ----        | ----    |
v-model  | Boolean | false |  控制组件的显隐，true为显示。
maskCloseable| Boolean | false | 控制点击蒙层是否可关闭v-action-box

## Methods
方法名称   |    说明    |    参数    |
----    | ----      | ----        |
hide | 关闭 Action-box 组件 | 无
show | 打开 Action-box 组件 | 无
