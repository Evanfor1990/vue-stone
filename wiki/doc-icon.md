# Icon

Icon 包含了常用的图标；

## Tag Name
`v-icon`

## Usage
```html
<v-icon
  name="icon-large-volume"
  size="12px"
  color="#000"
></v-icon>
```

## Options

| 属性名 | 类型 | 默认值 | 说明 |
| ---- | ---- | ---- | ---- |
| name |  String  |  '' | 设置icon的图标名称 |
| size | String | '' | 设置图标的大小，例如：'12px' |
| color | String | $c-primary | 设置图标的颜色，默认是主题色，例如：'#000' |
