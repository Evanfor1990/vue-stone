# Picker

## Tag Name

`v-picker-loop`

## 6.x
属性名   |    类型    |    默认值    |   说明
----    | ----      | ----        | ----    |
type | String | line | line-不循环，circle-循环。
value | String | 无 |初始化当前选中的值。
unit | String | '' | 单位。
list-data | Array | [] | 选项列表。

## Events
方法名称   |    说明    |    参数    |
----    | ----      | ----        |
input | 滚动完成后会触发此事件 | 当前被激活选项所组成的选项的值。
