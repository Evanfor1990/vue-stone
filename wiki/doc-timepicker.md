# Timepicer

一个时间选择器，只能选择 时 和 分。

## Tag Name
`v-icon`

## Usage
```html
 <v-icon name="icon-large-volume" size="12px" color="#f0f"></v-icon>
```

## Options
| 属性名 | 类型 | 默认值 | 说明 |
| ---- | ---- | ---- | ---- |
| name | String | '' | 设置icon的名字 |
| size | String | '' | 设置字体图标的大小，例如: '12rem' |
| color | String | $c-primary | 设置字体图标的颜色，默认是主题色, 例如: '#000' |
