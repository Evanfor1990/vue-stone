# Textarea

## Usage

```html
<v-textarea maxlength="100" showCounter placeholder="请输入" ></v-textarea>
```

## Options
属性名   |    类型    |    默认值    |   说明
----    | ----      | ----        | ----    |
maxlength | Number | - | 最大输入字数
placeholder | String | - | 初始提示文字
readonly | Boolean | false | 是否只读
showCounter | Boolean | false | 是否显示计数器	
callback | Function	 | - | 更改后调用的方法
